## 关键改进

| 问题 | 解决方案 |
|------|----------|
| 任务卡死无反馈 | 超时定时器自动终止 + 推送超时通知 |
| 长时间任务用户不知状态 | 心跳机制定期推送进度 |
| 命令失败静默停止 | 退出码检测 + 自动失败通知 |
| 需要手动配对 start/done | 单命令完成全部流程 |

## 使用方式

### 推荐方式（run 命令）
```bash
# 简单任务
async-task run --desc "统计文件" --cmd "find . -type f | wc -l" --timeout 60

# 长时间任务带进度
async-task run --desc "训练模型" --cmd "python train.py" --timeout 3600 --heartbeat 300
```

### 传统方式（向后兼容）
```bash
async-task start "分析中..."
# ... 执行命令 ...
async-task done "完成"
```
