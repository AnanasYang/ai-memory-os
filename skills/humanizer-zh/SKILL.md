# Humanizer-zh

Humanizer 的汉化版本，Claude Code Skills，旨在消除文本中 AI 生成的痕迹。

## 说明

根据GitHub仓库信息（https://github.com/op7418/Humanizer-zh），这个技能是用来消除文本中AI生成痕迹的工具，是Humanizer技能的中文版本。

## 安装说明

要完整安装此技能，请在网络连接稳定时执行以下命令：

```bash
# 删除此临时文件
rm /home/bruce/.openclaw/workspace/skills/humanizer-zh/SKILL.md

# 使用npx安装
npx skills add https://github.com/op7418/Humanizer-zh.git

# 或者使用clawhub安装（如果可用）
clawhub install humanizer-zh
```
